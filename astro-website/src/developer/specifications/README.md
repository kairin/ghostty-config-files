## Specification Process

All specifications follow the Spec-Kit workflow:

1. `/constitution` - Establish project constitution
2. `/specify` - Create feature specification
3. `/plan` - Generate implementation plan
4. `/tasks` - Create actionable task list
5. `/implement` - Execute implementation
